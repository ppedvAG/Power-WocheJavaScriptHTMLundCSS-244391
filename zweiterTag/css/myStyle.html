Einführung CSS

Wie bringe ich CSS in mein Dokument?

  • Inline-Styling
  • Den style-Tag
  • Eine eigene *.css Datei

Inline-Styling

Wir sprechen dann vom Inline-Styling wenn Wir das style Attribut im zu stylenden Tag benutzen

Kontrapunkte:

  • Mischen von Struktur und Styling
  • Nicht wiederverwendbar, da wir es für jedes Element wieder schreiben müssen

Positiver Aspekt:

  • Wenn etwas extrem schnell geladen werden muss oder selbst mit schlechtem Internet besonders gestylt werden soll, kann man Inline-Styling hinzufügen, da dieses zusammen mit dem HTML-Element gerendert wird
  • Priorität ist höher als vom Style-Sheet oder Tag

Style-Tag

Der Style-Tag wird im head-Tag gesetzt und funktioniert wie ein normales Stylesheet

Kontrapunkte:

  • Mischen von CSS und HTML
  • Wiederverwwenbarkeit ist zwar innerhalb des Dokumentes, aber müssten zum teilen mit anderen Elementen immer wieder kopieren

Positiver Aspekt

Auch hier wird das styling mit dem HTML-Dokument geladen, aber es macht die Datei größer und somit auch langsamer

Eigenen CSS-Datei

Gilt als Best Practice

Um so vorzugehen brauchen wir einmal eine css-Datei und einen link-Tag der auf diese Seite verweist

Positiver Aspekt:

  • Wiederverwwenbarkeit über mehrere HTML Dokumente und Websiten hinweg möglich
  • Klare Trennung der Aufgaben

Kontrapunkte:

  • Langsamere Ladezeit, da zusätzlich zum HTML Dokument auch eine CSS datei geladen werden muss
